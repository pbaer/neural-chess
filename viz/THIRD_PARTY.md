# Third-party assets

## Chess pieces — "cburnett"

The chess piece graphics rendered by the play board and the model-inspector
boards are the **cburnett** vector set by **Colin M.L. Burnett**. They are the
standard piece set used on Wikipedia and Lichess.

- **Author:** Colin M.L. Burnett (User:Cburnett)
- **License:** BSD (the set is multi-licensed GFDL / BSD / GPLv2+; this project
  uses it under the **BSD** license, with attribution).
- **Source:** Lichess `lila` repository,
  <https://github.com/lichess-org/lila/tree/master/public/piece/cburnett>
  (originally published on Wikimedia Commons as `Chess_<piece>45.svg`).

The SVG path data is embedded inline in
`src/presentations/desktop/play/pieces.tsx`.
